# UMF Taxonomy Artifacts

This directory contains independently versioned controlled-vocabulary artifacts.

Files:

- `ingredient-taxonomy.v1.0.0.json`
- `dietary-presets.v1.0.0.json`

## Contract

- Taxonomy artifacts are normative inputs for deterministic category and dietary reasoning.
- Consumers should pin exact taxonomy versions.
- Changes to existing leaf semantics require a major taxonomy version.

## Governance

See [ADR-003](../docs/adr/ADR-003-taxonomy-versioning-and-vocabulary.md).
